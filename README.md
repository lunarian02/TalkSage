# TalkSage

AI conversations with history's greatest thinkers.

TalkSage is an AI mentor platform that helps users reflect on life, decisions, and personal challenges through conversations with AI personas inspired by influential philosophers, writers, and teachers.

> This repository contains the public product concept, roadmap, and design notes for TalkSage. The application code is maintained separately.

## Concept

Most AI assistants provide answers.

TalkSage focuses on helping users think.

Each persona offers a different perspective, helping users explore the same question through multiple philosophical and emotional lenses.

## Personas

- Socrates
- Marcus Aurelius
- Seneca
- Laozi
- Nietzsche
- Oscar Wilde
- Modern Monk

## Example Questions

- How should I handle uncertainty?
- How do I recover from failure?
- What is a meaningful life?
- How should I make difficult decisions?
- How do I stop comparing myself to others?

## Vision

A personal council of advisors available anytime.
